CQFWord
===========
## 说明

    基于eletron写的电脑版背单词软件，支持中英文查询

## 环境配置与开发

#### 1、环境依赖

    首先，若没有gulp全局依赖，则需要先安装gulp全局依赖，若已经安装则直接跳过
    
        npm install -g gulp
        
    接着，直接在项目文件下执行 npm install 即可
  
#### 2、执行

    确保环境依赖执行正确以后，直接在项目文件下输入 gulp
  
#### 3、打包
 
    1.gulp move # 会多出一个dist的文件夹
  
    2.cnpm run packager # 多出一个打包完成的文件夹
